# PRODIGY_DS_01

## Task 1: Population Distribution Visualization

This project analyzes population data from the World Bank dataset and visualizes the distribution of the top 10 most populous countries in 2024 using a bar chart.

The dataset was cleaned and processed using Pandas, and the visualization was created using Matplotlib. The chart provides a clear comparison of population sizes among major countries, helping to understand global population distribution patterns.

### Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

### Dataset Source

World Bank Population Dataset
